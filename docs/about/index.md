# About VanillaPlus

**VanillaPlus** is a Multiplayer Server for Minecraft Java Edition {{ mc_version }}.  
It aims at providing the vanilla gameplay experience, enhancing it with new custom Blocks, furnitures, items, mechanics and more, all while remaining 100% vanilla client compatible.

## How it works

The Server utilizes plugins for its additional functionalities, with the [VanillaPlus](https://github.com/Andre601/VanillaPlus){ target="_blank" rel="noopener" } plugin being self-made.  
The resource pack alongside custom block, furniture, entity and item functionalities are handled and provided through [ItemsAdder](https://itemsadder.com){ target="_blank" rel="nofollow" } and [ItemsAdderAdditions](https://itemsadderadditions.com){ target="_blank" rel="nofollow" }.
