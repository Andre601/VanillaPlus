# Lava Fishing

**Lava Fishing** is a unique mechanic, allowing you to fish in Lava in the Nether Dimension using a [[Iron Fishing Rod]].

## How it Works

Lava Fishing works similarly to fishing in the overworld.  
The only difference is that instead of fishing in Water, you're fishing in Lava.

This mechanic only works in the Nether dimension and only with the Iron Fishing Rod.
