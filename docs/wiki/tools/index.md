# Tools

VanillaPlus provides a Set of tools, such as the [[Scythe]].

## Pages

/// html | div.grid.cards
-   ### [[Iron Fishing Rod]]
    
    Iron Fishing Rod is an upgraded variant of the normal Fishing Rod, providing higher durability.

-   ### [[Mortar]]
    
    Mortar is an item that can be used to craft [[mc:Bone Meal]], [[mc:Dyes]] and other selective items with twice the output they usually yield in their vanilla recipe.

-   ### [[Recipe Book]]
    
    Recipe Book is an item that opens the recipes GUI.

-   ### [[Scythe]]
    
    Scythe is a tool that allows you to harvest multiple crops simultaneously.

-   ### [[Teleport Crystal]]
    
    Teleport Crystal is an item that allows you to save a location to later teleport back to.

-   ### [[Totem of Experience]]
    
    Totem of Experience is an item that upon a player's death, stores their XP, allowing it to be retrieved to regain all their lost XP.
