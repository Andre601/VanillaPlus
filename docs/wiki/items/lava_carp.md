# Lava Carp

{{ infobox("vanillaplus:lava_carp") }}

**Lava Carp** is a consumable item that can be obtained when fishing in Lava Lakes in the Nether using the [Iron Fishing Rod](../tools/iron_fishing_rod.md).

## Usage

Lava Carp can be eaten, restoring 6 hunger and 4 saturation.

## Obtaining

### Fishing

Using an Iron Fishing Rod, Lava Carp can be obtained by fishing in Lava while in the Nether.
