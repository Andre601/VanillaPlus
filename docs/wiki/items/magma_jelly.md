# Magma Jelly

{{ infobox("vanillaplus:magma_jelly") }}

**Magma Jelly** is an item that can be obtained when fishing in Lava Lakes in the Nether using the [Iron Fishing Rod](../tools/iron_fishing_rod.md).

## Usage

Magma Jelly can be crafted into Magma Cream when put into the Crafting table, giving 1 Magma Cream per Magma Jelly.

## Obtaining

### Fishing

Using an Iron Fishing Rod, Magma Jelly can be obtained by fishing in Lava while in the Nether.
