# Couch

{{ infobox("vanillaplus:acacia_couch") }}

**Couch** is a furniture that can be placed and allow players to sit on it.
When placed next to each other will the benches of the same type connect similar to Stairs in Minecraft.

The item exists in all 12 wood types and all 16 wool color variants, allowing over 192 individual couch variants to be crafted.

## Usage

After placing the item on the floor can a player sit on it by right-clicking it.  
Holding the sneak button while sitting will dismount the player from the couch.
